Principais Formatações

Negrito / Destaque:

Nessa frase, temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase, temos um termo em destaque usando um <strong> (semântica).

Itálico / Ênfase:

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Texto marcado:

Podemos criar também um texto marcado usando a tag <mark>.

E no outro parágrafo, temos outro texto marcado no final

Estamos criando um texto grande com <big> e um texto pequeno com <small> nesse parágrafo.

Texto deletado:

Podemos marcar um texto como excluído, com a tag <del>, para indicar que ele deve ser lido, mas não considerado.

Texto inserido ou sublinhado:

Podemos marcar um texto como inserido, com a tag <ins>, para dar uma ênfase e indicar que ele foi adicionado depois (tag semântica para sublinhado).

Existe também o sublinhado com a tag <u> (não semântica).

Texto sobrescrito:

Para inserir coisas do tipo x20+3 em vez de x20+3, usar a tag <sup>.

Texto subscrito:

Para inserir coisas do tipo H20 em vez de H20, usar a tag <sub>.
